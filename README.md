# grandPy-Bot

**Présentation**
  > _Grandpy est une applications web permettant recherchez un lieu et obtenir des information le concernant._
  
 **Pré-requis**
  > _Une simple connection internet_ 
  
**Installation**
  > _L'installation ce fait en seulement 3 étapes:_
  
    *Télécharger le code source depuis le dépot
    *Télécharger les dépendances contenu dans le requirement.txt a l'aide de pip
    *Rendre Executable si nécessaire le fichier "run.py"
    
 **Exécution**
   > _Pour lancer l'application rien de compliqué:_
    
     *Lancer run.py
     *Ouvrir le lien donner par l'application dans un navigateur
		
 **Utilisation**
  > _L'utilisation de l'application en elle même est très simple, il vous suffit d'entrer une adresse(ex: paris) dans la barre de chat et grandPy s'occupe du reste._
